# PawPal 项目简介

PawPal 是一个面向高桥宠物友好村的 NFC 导览 MVP。游客通过 NFC 标签或模拟入口打开不同点位，在页面中遇见 PawPal，了解点位故事，并通过简单互动完成探索任务。

项目的第一阶段目标不是做完整产品，而是搭建一个清晰、可扩展的前端骨架，让后续可以逐步接入 UI 参考图、PawPal 形象、音频素材和真实点位内容。

## 核心体验

- NFC 或 URL 模拟进入点位。
- PawPal 作为导览伙伴出现。
- 页面展示点位介绍和轻量互动。
- 完成指定动作后解锁猫爪图鉴。

## MVP 范围

当前阶段只创建项目骨架、文档和素材目录。后续再实现完整交互、动画、音频和图鉴状态。
