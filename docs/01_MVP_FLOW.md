# 最小闭环流程

## 用户路径

1. 用户通过 NFC 或模拟按钮打开点位。
2. 页面读取 URL 中的点位参数，例如 `?point=gate`。
3. PawPal 出现在点位页。
4. 用户阅读点位介绍。
5. 用户点击「听听这里」。
6. 用户点击「摸摸 PawPal」。
7. 两个动作都完成后，自动弹出「完成任务」弹窗。
8. 用户解锁「高桥宠物友好村猫爪图鉴」。
9. 用户进入图鉴页，看到对应点位的猫爪已点亮。

## 第一版实现原则

- 先保证路径清晰，再增加动画和视觉细节。
- 先用文字模拟音频播放。
- 先用本地数据模拟点位和解锁状态。
- 暂不实现浏览器 Web NFC 读取或写入。
- 真实落地时，NFC 标签写入点位 URL，例如 `https://example.com/?point=cat-island`。
- 当前 URL 参数模拟的是未来真实 NFC 标签打开页面后的结果。
